# Travel Planner & Booking Hub - Replit Project

## Overview
A React-based travel planning and booking website featuring a responsive design with components for navigation, home hero section, main destination listings, and footer. Built as an academic project demonstrating modern frontend development practices.

## Project Type
- **Frontend Only**: React single-page application
- **Framework**: Create React App (react-scripts)
- **Styling**: SCSS with component-scoped styles

## Tech Stack
- **React** 18.2.0
- **React Icons** for UI icons
- **AOS** (Animate On Scroll) for scroll animations
- **Framer Motion** for advanced animations
- **Lucide React** for modern icons

## Project Structure
```
src/
├── Assets/          # Images and videos
├── components/
│   ├── Navbar/     # Navigation component
│   ├── Home/       # Hero section
│   ├── Main/       # Main content/destinations
│   └── Footer/     # Footer component
├── App.js          # Main app component
└── index.js        # Entry point
```
